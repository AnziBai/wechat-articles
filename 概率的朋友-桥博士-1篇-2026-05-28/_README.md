## 写作骨架

**骨架 A：诊断式**

| 步 | 落点 |
|---|---|
| 开篇 | 公式 B 行为诊断题——"你上一次买股票，是先看了大盘趋势再下手，还是看到群里有人喊'这只票要起飞'就跟进去的？" |
| 职业 vs 业余对比 | 公募经理晨会动作 vs 散户开盘看持仓涨跌 |
| 命名差距 | **散户输给机构的差距，不是信息，是结构**（三层结构：大盘判断 + 相对强弱 + 仓位止损） |
| 心理学根因 | 几十万年进化下来对"看起来确定"的偏好——不是懒不是笨 |
| 量化时刻 | 追高 10% + 止损 20% = 单笔 30% 摩擦；下一笔需盈利 43% 回本；死扛跌 50% 则需盈利 100% |
| 宽论解法 | 一个中心 + 两个基本点 + 三个工具；着重讲弹论（5/10/20/40/60 日均线切弹簧上中下三态） |
| 收尾翻转定义 | "散户的概率优势从来没消失，消失的是结构。把结构找回来。" |
